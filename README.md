# KWWD-Simple-Slider
A WordPress plugin that shows a simple scrolling image carousel vias a shortcode

[![WordPress Version](https://img.shields.io/badge/wordpress-%3E%3D6.0-blue.svg)](https://wordpress.org)
[![License](https://img.shields.io/badge/license-GPLv2-green.svg)](LICENSE)
[![PHP Version](https://img.shields.io/badge/php-%3E%3D8.0-8892bf.svg)](https://php.net)

**Simple Slider by KWWD** - an image slider carousel plugin for WordPress.

The plugin allows you to create multiple sliders, set global or per slider variables, upload images with captions and auto-generate trackable shortlinks if you have the URL Shortify plugin installed.

The Simple Slider by KWW WordPress Plugin allows you to easily create Responsive carousels and you can easily display multiple responsive carousel sliders where ever you can use shortcodes and this plugin is perfect for displaying products, Amazon affiliate links, photos and more.

If you have URL Shortify installed then a shortlink is automatically created and applied to the slide to enable click tracking.

A CSV bulk upload function is now also available to allow you to upload multiple slides to each slider. 

**Note**: You need to create the slider first before going in to the "Edit" page to be able to access the bulk upload. Images referenced in the CSV file should be the full image URL and images in the bulk upload CSV need to be uploaded via the media library, the plugin does not handle bulk image uploads.

The plugin is in active development and further functionality is already planned
